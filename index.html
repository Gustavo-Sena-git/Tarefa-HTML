Teg <ul>

Esta função é usada para criar lista não ordenadas, ou seja com marcadores para cada item da lista. Exemplo:
  • Maçã
  • Banana
  • Melancia

Teg <ol>

Usada para criar lista numeradas. Exemplo:
  1. Maçã
  2. Banana
  3. Melancia

Teg <li>

Usada para criar itens dentro de listas. Exemplo:
  1. Maçã
  2. Banana
  3. Melancia
